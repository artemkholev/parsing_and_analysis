# Processing Pipeline

## Requirements

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

```bash
python3 app.py path/to/hh.csv
```

This will create `x_data.npy` and `y_data.npy` in the same directory as the input CSV file.

## Example

```bash
python3 app.py example/hh.csv
```
